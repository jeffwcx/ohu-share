# ohu-share

跨浏览器的分享插件，可优雅降级

## 分享策略

该分享插件秉持稳定，可靠的特点，可在PC端和移动端分别进行分享，移动端如果目标分享对象支持原生调用分享，那就进行原生分享，如果不支持，使用URL分享；PC端则一贯使用URL分享。

![flowchart](./assets/flowchart.png)

**分享目标**

+ 微信好友
+ 微信朋友圈
+ QQ好友
+ QQ空间
+ 微博

**移动端浏览器支持列表**

+ QQ浏览器
+ UC浏览器
+ 360浏览器
+ 百度浏览器
+ 搜狗浏览器
+ 猎豹浏览器
+ 2345浏览器
+ Chrome浏览器
