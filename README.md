## 分享策略

该分享插件秉持稳定，可靠的特点，可在PC端和移动端分别进行分享，移动端如果目标分享对象支持原生调用分享，那就进行原生分享，如果不支持，使用URL分享；PC端则一贯使用URL分享。

![flowchart](./assets/flowchart.png)

**分享目标**

✅ 微信好友

✅ 微信朋友圈

✅ QQ好友

✅ QQ空间

✅ 微博

**移动端浏览器支持列表**

✅ UC浏览器

✅ 百度浏览器
